## How to make files accessible only by root?
    
    - https://askubuntu.com/questions/193055/how-to-make-files-accessible-only-by-root

    ```bash
    sudo chown root:root /path/to/application
    ```

    ```bash
    sudo chmod 700 /path/to/application
    ```
