# Course Introduction Summary

### Course Overview

This course provides an overview of the benefits of certification, the structure of the course, and everything you need to help you pass the AZ-900 exam.

### [Azure Cloud Shell](./cloud-shell.md):

- **Works from anywhere:** using a browser or mobile app.
- **Many included tools and features:** including storage.

### [Azure Portal](./azure-portal.md)

An intuitive, visual one-stop shop for managing all your Azure resources.

### [Azure CLI](./azure-cli.md)

A text-only way to manage Azure resources. Provides a stable, structured way to manage your Azure environment.

### [Azure Mobile Apps](./azure-mobile-apps.md)

Use the Android and iOS mobile apps to be alerted and respond to emergencies, or just perform general maintenance.

### [ARM Templates](./arm-templates.md)

Use ARM templates to automate your infrastructure setup and avoid human mistakes. They are idempotent, can be put into source control, and are declarative.

### [Azure PowerShell](./azure-powershell.md)

Fully supported by Azure with modules and cmdlets. A tool that fits into many different workflows.
