## Contents Overview

This module provides a foundational understanding of Azure, its key management tools, and concepts. It covers various ways to interact with Azure, including the Azure Portal, CLI, PowerShell, and Cloud Shell, along with an introduction to ARM Templates and Azure Mobile Apps.

### Files

-   [arm-templates.md](./arm-templates.md): Explains Azure Resource Manager (ARM) templates for infrastructure as code.
-   [azure-cli.md](./azure-cli.md): Notes on using the Azure Command-Line Interface.
-   [azure-mobile-apps.md](./azure-mobile-apps.md): Information about managing Azure resources using mobile applications.
-   [azure-portal.md](./azure-portal.md): Overview of the Azure web-based portal.
-   [azure-powershell.md](./azure-powershell.md): Notes on using Azure PowerShell for resource management.
-   [cloud-shell.md](./cloud-shell.md): Details about the browser-based Azure Cloud Shell.

## Role in System

This module is the starting point for anyone learning Azure, providing essential knowledge about how to navigate and manage Azure resources effectively. It lays the groundwork for understanding more advanced Azure services and architectures.

## Course Introduction Summary

### Course Overview

This course provides an overview of the benefits of certification, the structure of the course, and everything you need to help you pass the AZ-900 exam.

### Key Azure Tools and Concepts

-   **[Azure Cloud Shell](./cloud-shell.md)**: A browser-based shell for managing Azure resources.
    -   Works from anywhere using a browser or mobile app.
    -   Includes many tools and features, including storage.

-   **[Azure Portal](./azure-portal.md)**: An intuitive, visual one-stop shop for managing all your Azure resources.

-   **[Azure CLI](./azure-cli.md)**: A text-only way to manage Azure resources, providing a stable, structured method.

-   **[Azure Mobile Apps](./azure-mobile-apps.md)**: Use the Android and iOS mobile apps to be alerted and respond to emergencies, or perform general maintenance.

-   **[ARM Templates](./arm-templates.md)**: Use ARM templates to automate your infrastructure setup and avoid human mistakes. They are idempotent, can be put into source control, and are declarative.

-   **[Azure PowerShell](./azure-powershell.md)**: Fully supported by Azure with modules and cmdlets. A tool that fits into many different workflows.
