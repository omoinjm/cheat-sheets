## Parent Context

This directory is part of the "Compute Services" module within the AZ-900 Microsoft Azure Fundamentals course, serving as a repository for visual aids.

## Contents Overview

This directory stores various image files that are used to visually illustrate concepts and features discussed in the `04 - Compute` module. These images enhance understanding of Azure's compute services such as Virtual Machines, App Services, Container Instances, Kubernetes, and Functions.

### Files

-   `aci-features.png`: Image depicting features of Azure Container Instances (ACI).
-   `aks-scenario-add-pod.png`: Image illustrating a scenario of adding a pod in Azure Kubernetes Service (AKS).
-   `aks-scenario.png`: Image depicting a general scenario in Azure Kubernetes Service (AKS).
-   `aks.png`: Image related to Azure Kubernetes Service (AKS).
-   `api-apps.png`: Image illustrating API Apps within Azure App Services.
-   `avd-benefits.png`: Image highlighting benefits of Azure Virtual Desktop.
-   `azure-functions.png`: Image related to Azure Functions.
-   `azure-virtual-desktop.png`: Image related to Azure Virtual Desktop.
-   `function-benefits.png`: Image depicting benefits of Azure Functions.
-   `kubernetes.png`: Image related to Kubernetes.
-   `scale-sets-benefits.png`: Image highlighting benefits of Azure Virtual Machine Scale Sets.
-   `use-cases.png`: Image illustrating use cases for Virtual Machines.
-   `vm-features.png`: Image depicting features of Azure Virtual Machines.
-   `web-apps-containers.png`: Image illustrating Web Apps for Containers within Azure App Services.
-   `web-apps.png`: Image illustrating Web Apps within Azure App Services.

## Role in System

These images enhance the clarity and user-friendliness of the accompanying text-based documentation. They provide visual context, making abstract Azure compute service concepts and their functionalities easier to grasp for the reader and improving the overall learning experience for the AZ-900 course.
