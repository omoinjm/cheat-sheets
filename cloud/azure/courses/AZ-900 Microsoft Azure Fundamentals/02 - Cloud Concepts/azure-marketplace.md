# Azure Marketplace

1. **Solutions and Services**
    
    Large selection from Microsoft and partners. Apps, VMs, templates, services and oh so much more.
    
2. **Azure App Store**
    
    Buy cloud services with a single click (almost). Many categories of items to acquire.
    
3. **Easy to Integrate**
    
    Use from Portal, CLI or PowerShell. Some are free, some are paid.
    
4. **Publish Your Own**
    
    If you are a Microsoft Partner, publish your own services in the marketplace.
