<h2> üìö Cloud Concepts Summary</h2>

### [[language-cloud-computing.md|Language of Cloud Computing]]
Covers terms like high availability, fault tolerance, disaster recovery, scalability, elasticity, and agility. Describes stable and dependable cloud computing, the ability to adapt to changes in resource demand, user base, and application usage.

### [[language-cloud-economics.md|Language of Cloud Economics]]
Discusses CapEx and OpEx as costs for computing. OpEx is a key characteristic of cloud computing with its pay-as-you-go model.

### [[cloud-service-models.md|Cloud Service Models]]
Introduces IaaS, PaaS, and SaaS, which are the fundamental cloud service models encompassing nearly all Azure products and services. The shared responsibility model dictates whether you or Microsoft is responsible for a cloud service.

### [[azure-marketplace.md|Azure Marketplace]]
An extra layer of functionality for your cloud applications, allowing users to integrate third-party products and services.

### [[cloud-architecture.md|Cloud Architecture Models]]
Explores private, public, and hybrid approaches to using cloud computing for your business.
