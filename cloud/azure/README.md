## Contents Overview

This directory contains documentation and resources related to Microsoft Azure cloud services. It covers topics such as Azure Active Directory (AAD), Azure Container services, and learning materials from Azure certification courses.

-   [courses](./courses/README.md): Structured learning materials and notes for various Azure certification courses (e.g., AZ-900).

## Role in System

This section serves as a comprehensive knowledge base for individuals working with Microsoft Azure. It provides practical guides for configuring Azure services, managing cloud resources, and preparing for Azure certifications, thereby supporting cloud development and operations.
