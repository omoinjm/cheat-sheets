## Parent Context

This directory is part of the Hack The Box (HTB) notes, specifically storing VPN configuration files required for connecting to HTB labs.

## Contents Overview

This directory contains OpenVPN configuration files (`.ovpn`) necessary to establish a VPN connection to the Hack The Box platform. These files enable access to the lab environments where HTB machines are hosted.

### Files

-   `academy.ovpn`: OpenVPN configuration file for connecting to the HTB Academy platform.
-   `start_point_omoi.ovpn`: OpenVPN configuration file for connecting to the HTB Starting Point labs, potentially specific to a user or session.

## Role in System

These VPN configuration files are critical entry points for engaging with the Hack The Box learning environment. They allow users to establish a secure connection from their local machine to the HTB network, enabling participation in challenges and machine compromises.
