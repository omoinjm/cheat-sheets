# Notes 

### NMAP scan

  -  ```sudo nmap -sS -sV <ip address>```

### Connecting to different protocols/services

  - telnet: 
    - port 22/tcp
    -  ```telnet <ip address>```
    
  - ftp: 
    - port 21/tcp 
    - ```ftp <ip address>```
    
  - smb:
    - port 445/tcp
    - ```smbclient //ip address/Sharename```
