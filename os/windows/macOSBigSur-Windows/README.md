## Contents Overview

This directory contains a collection of cursor files (`.cur`, `.ani`) designed to replicate the macOS Big Sur cursor theme on Windows operating systems. It also includes an `install.inf` file for easy installation.

### Files

-   `*.cur`, `*.ani`: Cursor files for various states (e.g., default, busy, text select).
-   `install.inf`: An information file used by Windows to install the cursor theme.

## Role in System

This collection allows users to customize their Windows desktop experience by applying the visual style of macOS Big Sur cursors, contributing to aesthetic personalization.

## Installation

To install these cursors, right-click on the `install.inf` file and select "Install". After installation, you can select the "macOSBigSur" cursor scheme from the Mouse Pointers settings in the Control Panel.
