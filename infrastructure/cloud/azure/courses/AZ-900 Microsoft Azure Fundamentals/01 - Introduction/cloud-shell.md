## Contents Overview

This file introduces the Azure Cloud Shell, an interactive, browser-accessible shell environment for managing Azure resources. It highlights its key features, including accessibility, choice of shell, pre-installed tools, persistent storage, and an integrated file editor.

## Role in System

Azure Cloud Shell provides a convenient and powerful command-line interface directly within the Azure Portal or as a standalone service. This document explains its advantages for administrators and developers who need to execute scripts, manage resources, and troubleshoot issues without installing local tooling.

## Shell Experience

Cloud Shell is an interactive, browser-accessible shell for managing Azure resources.

-   **Stand-alone**: Available as a browser version.
-   **In-Portal**: Integrates as a portal component, offering a Bash experience similar to Azure CLI.

### Features

-   **Access**: Access from anywhere using a web browser or mobile app. It is authenticated and secure.
-   **Shell**: Choose between Bash (for Azure CLI) or PowerShell.
-   **Tools**: Comes with pre-installed tools, interpreters, modules, and Azure tools. It includes language support for Node.js, .NET, and Python.
-   **Storage**: Features dedicated, persistent storage to save data between sessions.
-   **File Editor**: Includes a complete file editor for quick modifications.
