## Contents Overview

This file introduces Azure Virtual Desktop (AVD), a desktop and app virtualization service that runs on Azure. It explains what AVD looks like, its compatibility with various VMs, accessibility features, and the benefits of using a fully virtualized Windows experience.

## Role in System

Azure Virtual Desktop provides a scalable and secure solution for delivering virtualized desktops and applications to users anywhere. This document highlights how AVD can simplify desktop management, enhance security, and enable remote work scenarios by centralizing the Windows desktop experience in Azure.

## Key Concepts and Features

### What Does it Look Like?

AVD runs entirely on Azure, providing a cloud-hosted Windows experience.

-   ![Azure Virtual Desktop Overview](./images/azure-virtual-desktop.png)

### Use Any VM You Want

-   AVD can run on any Windows Virtual Machine (VM) in Azure (Linux VMs are not supported for the host pool operating system).
-   Users can access their virtual desktops from any device that has a web browser and an internet connection.
